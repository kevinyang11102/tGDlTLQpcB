# 宠物管理系统简介 java368

## 项目概述

宠物管理系统是一套后台管理系统，主要用于流浪动物管理及救助工作，支持协同过滤推荐算法，可根据需求添加支付宝沙箱支付及AI模型对话功能。

## 技术栈

- 开发语言：Java
- 数据库：MySQL
- 后端框架：SpringBoot
- 前端框架：Vue
- 系统环境：Windows，Mac
- 开发环境：IDEA、JDK1.8

## 功能模块

### 前台功能

- **首页**
  - 公告列表
  - 宠物科普
  - 流浪宠物列表
  - 热门宠物
- **宠物领养**
  - 查看我的领养
  - 领养宠物列表
- **宠物救助**
  - 新增救助信息
  - 救助站
  - 宠物喂点
- **丢失宠物查看**
  - 查看宠物具体信息
- **宠物论坛**
  - 查看宠物论坛情况
- **个人中心**
  - 个人资料
  - 修改密码

### 后台功能

- **主页**
  - 欢迎进入系统
  - 查看数据表
- **宠物管理**
  - 流浪动物管理
  - 宠物绝育管理
  - 流浪动物救助
  - 宠物走失管理
  - 喂养点管理
  - 救助站管理
  - 申请领养管理
- **其他管理**
  - 帖子管理
  - 捐赠管理
  - 科普文章管理
  - 活动管理

## 系统角色

系统包含两类角色：前台用户与后台管理员。

- 前台用户：可进行宠物领养、信息查看、论坛互动等操作。
- 后台管理员：负责宠物管理、系统维护、数据监控等任务。
